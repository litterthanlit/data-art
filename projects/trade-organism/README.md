# Trade Organism

Abstract 3D data sculpture of the global supply chain as a living system.

Open locally:

```sh
npm install
npm run data
npm run dev
```

Then visit:

```text
http://127.0.0.1:5173/
```

Data sources:

- World Bank container port traffic indicator `IS.SHP.GOOD.TU`
- UN Comtrade API as the intended commodity-flow reference
- Public chokepoint references for Suez, Panama, Malacca, Hormuz, Gibraltar, and Singapore

The bundled JSON is the stable artwork cache. The scene is an artistic transformation, not a literal logistics dashboard.
