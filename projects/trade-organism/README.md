# Trade Organism

Abstract 3D data sculpture of the global supply chain as a living system.

Open locally:

```sh
npm install
npm run data
npm run dev
```

Then visit:

```text
http://127.0.0.1:5173/
```

Data references:

- World Bank container port traffic indicator `IS.SHP.GOOD.TU` as a scale reference
- UN Comtrade API as a commodity-flow reference
- Public chokepoint references for Suez, Panama, Malacca, Hormuz, Gibraltar, and Singapore

The bundled JSON is a stable illustrative artwork cache. It is curated from public supply-chain references and scaled from container-traffic patterns; it is not live fetched data or a literal logistics dashboard.
