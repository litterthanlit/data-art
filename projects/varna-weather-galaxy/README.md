# Varna Weather Galaxy

Interactive atmospheric data sculpture generated from 11 years of hourly Varna weather readings.

Open locally:

```sh
npm install
npm run dev
```

Then visit:

```text
http://127.0.0.1:5173/
```

Data source: Open-Meteo Historical Weather API.

The app ships with a bundled local weather cache at `public/data/varna-weather-2015-2025.json`.
